## Craw QCon 2016 files Console Tool


just run
```
git clone https://github.com/php-cpm/crawQCon.git
composer create-project laravel/laravel=5.1 bin
cd bin
cp -R crawQCon/ ./
composer update
./craw
```
find files under dir storage/app/files
